1. В файле `home.py` создайте логгер с именем `AnotherLogger`
2. Добавьте 3 обработчика:
    1. Выводит в консоль (`sys.stdout`) все сообщения уровня `logging.WARNING` и выше
    2. Выводит в файл `info.log` (оставляя старое содержимое) все сообщения уровня `logging.INFO` и выше
    3. Выводит в файл `debug.log` (затирая старое содержимое) все сообщения уровня `logging.DEBUG` и выше
3. Для первого обработчика добавьте форматирование вида `"%(asctime)s - %(levelname)s - %(name)s - %(message)s"`
4. Для третьего обработчика добавьте фильтр, который оставляет только сообщения уровня `logging.DEBUG`
5. Запустите скрипт `home.py`

**Результат**: Скриншот, на котором видны логи в консоли и каждом из файлов
